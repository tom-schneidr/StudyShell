Roadmap

Current state and next build priorities.

Baseline Complete

  • Native workspace browsing, file operations, live watching, and search.
  • Markdown, text, code, notebook, PDF, image, audio, and video workflows.
  • FreeRouter-backed chat, summaries, flashcards, quizzes, and source context.
  • Recent files, pinned files, split view, settings, command palette, and timer.

Next Priorities

  • Deepen tests around filesystem edge cases and editor state transitions.
  • Improve keyboard navigation and accessibility coverage across the shell.
  • Package a demo-ready release with screenshots and a short walkthrough.
  • Consider richer tab state once save/dirty handling is fully covered by tests.

Implementation Guardrails

  • Keep the UI clean and desktop-native.
  • Do not use blocking browser dialogs.
  • Keep new behavior behind typed hooks and testable utilities.
  • Clean up related app state when files are moved, renamed, or deleted.

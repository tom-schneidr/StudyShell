Architecture

React owns the experience. Rust owns privileged desktop work.

Frontend

The React app in src/ handles layout, editor state, viewers, command routing, AI interaction state, and persistence through focused hooks and utility modules.

Tauri Backend

Rust commands in src-tauri/src/ handle file I/O, directory watching, binary reads, search, imports, and the HTTP bridge to FreeRouter.

AI Boundary

StudyShell sends OpenAI-compatible requests to a local FreeRouter gateway. Provider keys stay outside the WebView and outside this repository.

Persistence

Lightweight app preferences live in localStorage behind named keys. Document-specific PDF annotations are saved as sidecar JSON files next to the source PDF.

Development Rules

  • Mirror new Tauri commands with typed frontend wrapper methods.
  • Validate file operations before mutating disk state.
  • Prefer pure utility functions for path, parsing, and state transforms so they remain testable.
  • Use existing CSS variables and components before introducing new design primitives.

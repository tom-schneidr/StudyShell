StudyShell

A local-first desktop workspace for serious study sessions.

StudyShell brings notes, code, PDFs, notebooks, media previews, search, flashcards, quizzes, and FreeRouter-backed AI assistance into one Tauri desktop app.

What Works Today

  • Workspace file browser with live refresh and native filesystem access.
  • Markdown, plain text, and CodeMirror-backed code editing.
  • PDF viewing, annotation sidecars, image/media previews, and notebook rendering.
  • AI chat, summarization, flashcards, and quizzes through local FreeRouter.
  • Recent files, pinned files, command palette, search, split view, and study timer.

Quick Start

npm install
npm run tauri:dev

AI features expect FreeRouter at http://127.0.0.1:8000/v1. Configure provider keys in FreeRouter, not inside StudyShell.

Verification

npm run verify

The full release check also includes npm run build and npm run tauri:build.
